Complete Setup Guide

Get up and running with DRAKON in minutes.

1 Download the Application

Get the standalone Windows executable for maximum performance and offline capability.

Download for x64
2 Install Ollama Engine

Ollama is the backend engine required to run local AI models. Install this first.

Get Ollama
3 Setup Hushiyar Model (Recommended)

For the best experience, we strongly recommend the Hushiyar model. Run these commands in your terminal:

1. Pull the model:

$ ollama pull aryanvala/hushiyar-alpha:latest

2. Run the model:

$ ollama run aryanvala/hushiyar-alpha:latest

Core Capabilities

Visual Intelligence

Integrated ComfyUI workflow ensures high-fidelity image generation. Unlike standard diffusers, DRAKON uses a custom node-based pipeline for "Juggernaut" quality visuals directly in chat.


Context Awareness

The system maintains a sliding window of chat history, ensuring the AI remembers previous code snippets and instructions without hitting token limits.


Secure Auth

Built on Firebase Authentication, providing enterprise-grade security for user accounts, session management, and history storage.


Reactive UI

A custom-built vanilla JS frontend that mimics the responsiveness of React, featuring real-time streaming, syntax highlighting, and glassmorphism.


Platform Native

Available as a standalone Windows executable (.exe) for maximum performance, offline capabilities, and seamless OS integration.


Advanced File Analysis

Upload documents, code files, and images directly into the chat. The AI engine automatically extracts and analyzes the content to provide highly accurate, context-aware responses.
